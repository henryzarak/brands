## Philosophy

1. **Vertical over horizontal** — go deep on fewer brands, not shallow on many
2. **Validate fast** — pivot or pause before sinking years
3. **All in-house** — software, content, ops. No SaaS lock-in.
4. **Build → preview → feedback → deploy** — nothing ships without local test
5. **B&W aesthetic** — visual consistency is a competitive moat
