# H Labs — Creative Studio

**Software · Dashboards · Automation · The studio behind the empire**

H Labs is the brand under which I build, ship, and maintain the software, dashboards, and automations that run Ondeska Holdings and its portfolio of brands.

## What we build

- **Holding dashboards** — unified view across all brands
- **Mission Control** — sprint + task board with auto-execution
- **OutreachCMD** — WhatsApp lead-gen pipeline
- **Brand dashboards** — per-brand metrics and KPIs
- **n8n workflows** — automation across the stack

## Stack

- **Frontend**: Next.js 15, vanilla HTML/CSS (B&W)
- **Backend**: FastAPI, Python 3.14, Node.js
- **Data**: SQLite, JSON vault
- **Infra**: Arch Linux, Tailscale, Docker, Caddy
- **Deploy**: Self-hosted on homelab

Every brand in the holding depends on H Labs software to operate. Building our own stack means no SaaS lock-in, no surprise pricing, full control over data.
